    0027. (link)

    Description

    Given an array nums and a value val, remove all instances of that value in-place and return the new length.
    Do not allocate extra space for another array, you must do this by modifying the input array in-place with O(1) extra memory.
    The order of elements can be changed. It doesn’t matter what you leave beyond the new length.

    27.1 Basic method

    Thought:
    Loop elements in nums, and if there is an element equal to val, then pop it.
    遍历整个列表，如果发现和val相等的元素，则删除。

    Time Complexity: O(N)


    Time Complexity: O(N)

    27.2 Fastest method

    Thought:
    Use extra variable index to indicate the length you want to return. Loop the list nums, if the item is equal to val, pass. If the item is not equal to val, then assign the item to nums[index], and index = index + 1.
    用一个变量来存储你最后要返回的值，就是nums去除之后的长度，初始化为0，然后遍历整个列表，如果和val相等，则遍历下一个，不相等则把这个值赋给nums[index]，然后index = index + 1.

    Time Complexity: O(N)

    Thought:
    Use two variablesleft, right to indicate the beginning and the ending of nums. If nums[left] equals to val, then swap nums[left] and nums[right], right += 1. If nums[left] doesnt’t equal to val, then left += 1. After loop, return right.

    用两个变量left, right 来记录nums的左端和右端。判断左端是否和val相等，相等则和右端交换，然后right += 1。不相等则left += 1。整体思路是把有效的（和val不相等的）放在左边，还没有检查的放在中间，无效的放在右边。最后返回right.
    Time Complexity: O(N)

    27.3 Shortest method

    Thought:
    Remove val until val is not in nums.
    Time Complexity: O(N^2)

    27.4 Summary

    1. a,b = b,a to swap values between two variables, you can also use a,b,c = b,c,a or something like that. The reason you can do this is Python will calculate the right of the =, and the right part is actually a tuple, and the assign the tuple to left.
      你可以用a,b = b,a的结构对两个变量的值进行交换，或者a,b,c = b,c,a的结构对多个变量的值进行交换。可以这样做的原因是，Python会首先处理=右边的部分，右边其实是由多个变量组成的一个元组，然后把元组中的值复制给左边。
