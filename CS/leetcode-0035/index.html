    0035. (link)

    Description

    Given a sorted array and a target value, return the index if the target is found. If not, return the index where it would be if it were inserted in order.

    You may assume no duplicates in the array.

    35.1 Basic method

    Thought:
    Loop the list from left to right, and find the first item which is larger than target. Then return the index of the item. If doesn’t find the item, then return the length of nums.
    从左到右遍历整个nums，找到第一个比target大的元素，然后返回这个元素的index。如果没有找到这个元素，则返回nums的长度。

    Time Complexity: O(n) (worst)

    35.2 Fastest method

    Thought:
    Use left and right to represent a closed interval for all possible position of target, and then use bisection algorithm to find the position. Find the median postion mid of [left,right], and then compare nums[mid] with target. If nums[mid] is equal to target, then return mid. If nums[mid] is larger than target, then assign mid to right. If nums[mid] is smaller than target, then assign mid+1 to left.
    用left和right来代表一个闭区间的左右两端，这个闭区间是所有target可能的位置，然后用二分法找到这个位置。首先找到闭区间[left,right]的中位数mid，然后比较nums[mid]和target。如果nums[mid]和target相等，则返回mid。如果nums[mid]比target大，则把mid赋值给right。如果nums[mid]比target校，则把mid+1赋值给left。最后闭区间的长度为1，返回left。

    Time Complexity: O(logN)

    27.3 Shortest method

    Thought:
    Append target to the nums, then sort the nums and return the index of target.
    在nums中插入target，然后对nums排序并返回target

    Time Complexity: O(NlogN)

    Thought
    Use bisect module to check the position of target.
    用bisect模块查找target的位置。

    Time Complexity: O(logN)

    27.4 Summary

    1. You can use bisect module to apply bisection algorithm.
      Python的模块bisect可以用来进行二分法操作
