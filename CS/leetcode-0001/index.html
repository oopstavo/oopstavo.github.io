    0001.Two Sum (link)

    Description

    Given an array of integers, return indices of the two numbers such that they add up to a specific target.
    You may assume that each input would have exactly one solution, and you may not use the same element twice.

    1.1 Basic method

    Thought:
    Loop the list nums, and try to find if sum of the item and one of following items equal to the target. If the sum is equal to the target, return index of these two items.
    遍历列表中的元素，找出该元素后面的元素是否和该元素的和为target。如果和为target，则返回二者的index。

    Time Complexity: O(0.5*n^2)

    1.2 Fastest method

    Thought:
    Try to create a dictionary stores the value:index pair, if the difference between target and an item is in the dictionary, then return index, if not then update the dictionary. This method is just loop nums once. And can skip the situation that one item equal to the difference between target and itself, since the first time come across the item, the difference(item itself) isn’t in the dictionary.
    声明一个字典变量，每次遍历nums的时候，检查在该字典变量中是否存在以target和这个元素的差值的键，如果存在则取出字典中键对应的值，连同遍历元素的index返回。如果不存在，则更新字典，以遍历元素的值为键，index为值。这种方法仅需要遍历nums一次，与第一种方法不同，这种方法是寻找遍历元素之前是否存在合适的值，不存在则更新字典，把之前遍历过的值以及index记录下来。

    Time Complexity: O(n)

    1.3 Shortest method (Code Golf)

    Thought:
    Try to find the difference in the remaining list, and return the index pair. This method can skip the situation that the difference is the same as the item.
    因为可以假设有且只有一个解，那么可以直接返回nums中元素和target与该元素差值的index。如果报错，pass后寻找下一个元素。
    Time Complexity: O(n^2)

    1.4 Summary

    1. enumerate(sequence, [start=0])is an iterator in Python, and the time complexity of enumerate(sequence, [start=0]) is O(1)
    2. Most of the dictionarty operations is O(1).
    Operation Example Complexity Class In-place or Not Notes
    Index d[k] O(1) N/A
    Store d[k] = v O(1) In-place
    Length len(d) O(1) N/A
    Delete del d[k] O(1) In-place
    get/setdefault d.get(k) O(1) N/A
    Pop d.pop(k) O(1) In-place
    Pop item d.popitem() O(1) In-place popped item “randomly” selected
    Clear d.clear() O(1) In-place similar to s = {} or = dict()

    Construction dict(…) O(len(…)) N/A depends # (key,value) 2-tuples
    Iteration for k in d: O(N) N/A all forms: keys, values, items
